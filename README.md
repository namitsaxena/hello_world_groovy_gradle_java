## Sample Project
Starter project with support for:
* Java
* Maven
* Groovy
* Gradle
* Junit
* Groovy Test Case
* Spock

## Building

### Maven
* mvn clean test

### Gradle
* gradle clean test 

## Requirements
The project can be run from maven, gradle or eclipse
* java 1.8
* mvn
* gradle
* groovy
* Eclipse
  * Groovy Eclipse
  * Gradle BuildSghip Plugin
